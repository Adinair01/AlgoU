# Git / GitHub Cheatsheet

Things I actually did in this assignment:

-> Created a public repository (AlgoU) with a README
-> Added a folder structure: docs/ and notes/
-> Created 3 issues to track tasks
-> Made a branch instead of editing main directly
-> Edited the README on that branch
-> Opened a Pull Request to merge the branch into main
-> Linked the PR to an issue using "Closes #1" so it auto-closed
-> Merged the PR and deleted the branch
-> Built a Projects board (Assignment Tracker) to track issue status
